# OzMoneyTracker - Project Instructions

## Overview
Local-only personal expense tracking app using expo-sqlite for persistence. No backend/API/auth.

## Tech Stack
- React Native + Expo (expo-router for navigation)
- expo-sqlite for local database (ozmoneytracker.db)
- expo-linear-gradient for gradient buttons/cards
- @expo/vector-icons (MaterialCommunityIcons)
- react-native-gesture-handler (Swipeable for swipe-to-delete)

## Design Theme
- Dark theme with deep navy/charcoal tones
- Primary: #4F46E5 (Midnight Indigo), Accent: #06B6D4 (Cool Cyan)
- Gradient buttons: ['#4F46E5', '#06B6D4']
- Background: #0D1117, Cards: #1C2128 with #30363D border
- Text: #E6EDF3 (primary), #8B949E (secondary), #6E7681 (muted)

## Navigation Structure
- Root Stack: tabs, add-expense (modal), edit-expense/[id], add-category (modal), edit-category/[id]
- Tabs: Dashboard, Expenses, Reports, Categories

## Database
- Tables: categories (id, name, color, isDefault, createdAt), expenses (id, name, amount, categoryId, date, createdAt, updatedAt)
- 5 default seeded categories: Alimentación, Transporte, Entretenimiento, Salud, Servicios
- Currency: USD ($), dates stored as ISO YYYY-MM-DD

## Key Files
- src/db/database.ts - All SQLite operations
- src/context/DatabaseContext.tsx - DatabaseProvider wrapping app
- src/theme.ts - Colors, spacing, radius constants
- src/components/ - MonthSelector, FAB, EmptyState, ExpenseRow, DatePickerField, ErrorBoundary
